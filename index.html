Python + geoai-py · Fields of The World (FTW)

Field boundary detection with instance segmentation (Andhra Pradesh)

This page documents the same end-to-end workflow as the Mask R-CNN + FTW tutorial: Sentinel-2 chips (RGB+NIR, 10 m), instance masks, training, inference, cleaning, and vector polygons. Training and inference run in a Python environment (not in the browser). For India-related work—including areas in Andhra Pradesh—use the FTW India subset; then clip or filter predicted vectors to your AP AOI using district or mandal boundaries.

Note (FTW India): The FTW benchmark includes an India country pack. Some documentation describes India (and a few other countries) as using presence-only or simplified labelling in parts of the benchmark. Check the latest FTW / TorchGeo notes for label semantics. For a small reproducible tutorial, you can still run the Luxembourg subset first, then repeat the same commands with country="india".

Why this matters

Accurate field boundary delineation supports precision agriculture, crop monitoring, subsidy verification, and land-use planning. Instance segmentation assigns a separate mask ID per field parcel, which is stronger than a single “all fields” mask.

1. Install

# pip install geoai-py # Use a virtual environment (recommended) python -m venv .venv # Windows: .venv\Scripts\activate # macOS/Linux: source .venv/bin/activate pip install geoai-py geopandas

2. Download FTW data (India for AP-focused work)

FTW provides 256×256 chips with four Sentinel-2 bands (R, G, B, NIR) and instance masks; many regions include two temporal windows (window_a / window_b) to exploit seasonal contrast.

import geoai geoai.download_ftw(countries=["india"], output_dir="ftw_data") # Smallest tutorial subset (optional first run): # geoai.download_ftw(countries=["luxembourg"], output_dir="ftw_data")

3. Explore chips (GeoParquet)

import os from pathlib import Path import geopandas as gpd country = "india" # or "luxembourg" country_dir = os.path.join("ftw_data", country) chips_gdf = gpd.read_parquet(os.path.join(country_dir, "chips_india.parquet")) # or chips_luxembourg.parquet print(len(chips_gdf), chips_gdf["split"].value_counts()) # Optional: filter chips intersecting an Andhra Pradesh bbox or your AOI shapefile # aoi = gpd.read_file("your_ap_aoi.geojson") # chips_gdf = chips_gdf[chips_gdf.geometry.intersects(aoi.union_all())]

4. Prepare training tiles

data = geoai.prepare_ftw("ftw_data", country="india") data

5. Train Mask R-CNN (instance segmentation)

num_classes=2 → background vs field. num_channels=4 → Sentinel-2 R,G,B,NIR. instance_labels=True → use FTW instance IDs in masks.

geoai.train_instance_segmentation_model( images_dir=data["images_dir"], labels_dir=data["labels_dir"], output_dir="field_boundaries/models", num_classes=2, num_channels=4, batch_size=4, num_epochs=20, learning_rate=0.005, val_split=0.2, instance_labels=True, visualize=True, verbose=True, )

6. Inference on a test chip

from pathlib import Path test_images = sorted(Path(data["test_dir"]).glob("*.tif")) test_image_path = str(test_images[0]) masks_path = "field_boundary_prediction.tif" model_path = "field_boundaries/models/best_model.pth" result = geoai.instance_segmentation( input_path=test_image_path, output_path=masks_path, model_path=model_path, num_classes=2, num_channels=4, window_size=256, overlap=128, confidence_threshold=0.5, batch_size=4, vectorize=True, class_names=["background", "field"], )

7. Clean instance mask → vector → properties

cleaned_masks_path = "field_boundary_prediction_cleaned.tif" geoai.clean_instance_mask( result["instance"], cleaned_masks_path, min_area=100, max_hole_area=100 ) output_vector_path = "field_boundary_prediction.geojson" gdf = geoai.raster_to_vector(cleaned_masks_path, output_vector_path) gdf_props = geoai.add_geometric_properties(gdf, area_unit="ha", length_unit="m")

Geometric properties (from GeoAI)

PropertyDescription
AreaField size (e.g. ha) — yield and subsidy context
PerimeterBoundary length (e.g. fencing)
ElongationMajor/minor axis ratio — strip vs compact parcels
SolidityArea / convex hull area — boundary irregularity
ExtentArea / bounding-box area — rectangularity

Andhra Pradesh: how to connect outputs to this app

  1. Train or fine-tune on FTW India chips; run inference on your own Sentinel-2 mosaics clipped to AP if needed.
  2. Export polygons as GeoJSON (or GeoPackage) in WGS84.
  3. Optional: clip polygons to mandal/district boundaries from your existing geojson_ap.geojson / ap_mandals.geojson.
  4. Replace or merge with data/chinagottigallu_fields.geojson or serve a new layer in the map views.

Repository script

A runnable template script is in scripts/ap_field_boundaries_ftw_geoai.py (download → prepare → optional train/inference). Run it from the project root after installing geoai-py. GPU is recommended for training.
